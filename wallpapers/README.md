# Wallpapers

Wallpaper assets used for local Linux workstation customization.

Expected structure:

wallpapers/
├── desktop-01.jpg
├── desktop-02.jpg
└── ...

These images can be used by future setup automation to configure the desktop background on a Linux workstation.
